Introduction
============

Installing
==========

You can install LoBot using::

    $ pip install lobot

Documentation
=============

View the latest LoBot documentation at: https://lobot.readthedocs.org.

Using
=====

Initialize LoBot by providing it with a wokring directory::

    $ mkdir lobot
    $ cd lobot
    $ lobot .
